# Mera İzleme Sistemi (MİS)

4342 sayılı Mera Kanunu kapsamında il/ilçe müdürlükleri için
BBHB (Büyükbaş Hayvan Birimi) hesaplama ve raporlama sistemi.

## Kurulum

```bash
npm install
cp .env.example .env   # ve içindeki değerleri düzenleyin
npm run dev
```

## Mimari

Proje yapısı ve tasarım ilkeleri için bkz. [`docs/MIMARI_REHBERI.md`](docs/MIMARI_REHBERI.md).

## Stack

Node.js 20, Express.js, Mongoose 8, MongoDB 7, Bootstrap 5, Vanilla JS,
ExcelJS, docx, pdfkit, Docker/Coolify.
